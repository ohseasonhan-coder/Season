# 사용자 입력 매뉴얼

## 1. 월 저축/투자 입력

예시:

```text
총액: 2,000,000원
비상금: 500,000원
ISA: 1,500,000원
```

저장 전 확인창에서 다음을 확인합니다.

- 분배 합계가 총액과 일치하는지
- ISA 연간 한도를 초과하지 않는지
- 비상금 목표가 부족한 상태인지
- 출금계좌와 입금계좌가 같은지

## 2. 비상금 입력

비상금은 KOFR, 파킹통장, 현금성 계좌 등으로 구분하세요.
비상금 계좌는 `includeInEmergency` 성격을 가진 항목으로 관리됩니다.

## 3. ISA 입력

ISA 입력은 연간 한도와 연결됩니다.
앱의 설정값이 실제 적용 한도와 맞는지 매년 확인하세요.

## 4. 카드 사용 / 카드 상환

카드 사용은 지출이지만 동시에 카드 부채 증가로 볼 수 있습니다.
카드 상환은 부채 감소로 처리하는 것이 좋습니다.

## 5. 백업 / 복원

복원은 즉시 반영하지 말고 비교 화면에서 현재 값과 후보 값을 확인한 뒤 진행하세요.
