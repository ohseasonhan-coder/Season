# 이번 v12에 포함된 내용

## 포함

- 사용자가 선호한 고기능 App.jsx 기반
- 로그인 한 글자 입력 후 포커스가 끊기는 문제 수정
- Supabase 로그인/동기화 구조 유지
- 로컬 저장 모드 유지
- 자동 백업/복원
- 거래내역
- 계좌/자산/부채
- 포트폴리오
- 예산
- 이벤트/목표
- ISA
- 연금/세금
- 투자 목표
- 자동 트리거
- CFO 분석
- 접이식 사이드바
- 모바일 로그인 시트
- Vercel 배포 구조
- public 폴더
- Electron 선택 실행 구조
- GitHub 전체 교체 가이드

## 제외 또는 주의

- 실제 결제 기능은 제외했습니다.
- 실제 상용 서비스 운영에는 보안 점검, Supabase RLS 정책, 약관/개인정보처리방침, 백엔드 권한 관리가 추가로 필요합니다.
- 이 패키지는 프론트엔드 중심의 상용 준비형 구조입니다.
